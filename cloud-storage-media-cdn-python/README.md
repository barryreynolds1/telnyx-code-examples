# Cloud Storage Media CDN

## What Does This Example Do?

Use Telnyx Cloud Storage as a CDN for IVR prompts, hold music, and voice assets. Categorized media library with direct playback URLs.

## Prerequisites

- Python 3.8+
- Telnyx account with API key from [portal.telnyx.com](https://portal.telnyx.com)

## Quick Start

```bash
git clone https://github.com/team-telnyx/telnyx-code-examples.git
cd telnyx-code-examples/cloud-storage-media-cdn-python
cp .env.example .env
# Edit .env with your credentials
make setup && make run
```

## Products Used

| Product | Role |
|---------|------|
| Cloud Storage | Media hosting |
| Voice API | IVR prompt delivery |
