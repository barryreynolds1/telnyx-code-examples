# AI Assistants Examples — Telnyx

Create, manage, and deploy LLM-powered assistants for voice and messaging automation.

## Best place to start

**[create-ai-assistant-python](../../create-ai-assistant-python/)** — Create an AI assistant and connect it to a phone number.

## What Telnyx AI Assistants gives you

- **Managed AI agents** — No LLM orchestration code needed
- **Built-in telephony** — Assistants answer phone calls natively
- **Knowledge bases** — Upload documents for RAG
- **Tool calling** — Connect assistants to your APIs
- **OpenAI-compatible** — Drop-in replacement for inference
