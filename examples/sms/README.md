# SMS & Messaging Examples — Telnyx

Send and receive SMS, MMS, WhatsApp, and RCS messages with the Telnyx messaging APIs.

## Best place to start

**[send-sms-python](../../send-sms-python/)** — Send your first SMS in 10 lines of code.

## What Telnyx Messaging gives you

- **SMS/MMS API** — Send and receive globally with delivery receipts
- **WhatsApp Business** — Programmatic WhatsApp messaging
- **RCS** — Rich card messaging with carousels and buttons
- **10DLC/Toll-Free** — Campaign registration and compliance
- **AI Inference** — AI-powered chatbots and auto-responders

## Production checklist

- [ ] Messaging Profile configured with webhook URL
- [ ] Phone number enabled for messaging
- [ ] 10DLC campaign registered (for US A2P)
- [ ] Opt-in/opt-out handling implemented
- [ ] Rate limiting and retry logic
