# IoT & SIM Management Examples — Telnyx

Activate SIMs, monitor data usage, provision eSIMs, track device locations, and build IoT alerting systems.

## Best place to start

**[activate-sim-card-python](../../activate-sim-card-python/)** — Activate a SIM card on the Telnyx network.

## What Telnyx IoT gives you

- **Global SIM cards** — Connectivity in 180+ countries
- **eSIM provisioning** — Over-the-air profile management
- **Data usage monitoring** — Real-time usage tracking and alerts
- **Device location** — GPS and cell-tower based tracking
- **Private networking** — VPN and private APN
