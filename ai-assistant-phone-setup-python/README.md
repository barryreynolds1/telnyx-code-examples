# AI Assistant Phone Setup

## What Does This Example Do?

Create and configure a managed Telnyx AI Assistant. Set voice, personality, greeting, model, and wire it to a phone number.

## Prerequisites

- Python 3.8+
- Telnyx account with API key from [portal.telnyx.com](https://portal.telnyx.com)

## Quick Start

```bash
git clone https://github.com/team-telnyx/telnyx-code-examples.git
cd telnyx-code-examples/ai-assistant-phone-setup-python
cp .env.example .env
# Edit .env with your credentials
make setup && make run
```

## Products Used

| Product | Role |
|---------|------|
| AI Assistants | Managed voice AI |
| Voice API | Phone number wiring |
| Inference | Model selection |
