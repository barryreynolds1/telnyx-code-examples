# Migrate from Vapi

## What Does This Example Do?

Import Vapi voice agents to Telnyx AI Assistants. Voice mapping, model translation, and prompt migration with one API call.

## Prerequisites

- Python 3.8+
- Telnyx account with API key from [portal.telnyx.com](https://portal.telnyx.com)

## Quick Start

```bash
git clone https://github.com/team-telnyx/telnyx-code-examples.git
cd telnyx-code-examples/migrate-from-vapi-python
cp .env.example .env
# Edit .env with your credentials
make setup && make run
```

## Products Used

| Product | Role |
|---------|------|
| AI Assistants | Agent import |
| Inference | Model mapping |
