# Media Stream Voice Cloak

## What Does This Example Do?

Real-time voice modification via media streaming API. Apply effects: deep voice, high pitch, robotic anonymization, or echo.

## Prerequisites

- Python 3.8+
- Telnyx account with API key from [portal.telnyx.com](https://portal.telnyx.com)

## Quick Start

```bash
git clone https://github.com/team-telnyx/telnyx-code-examples.git
cd telnyx-code-examples/media-stream-voice-cloak-python
cp .env.example .env
# Edit .env with your credentials
make setup && make run
```

| Product | Role |
|---------|------|
| Media Streaming | Real-time audio processing |
