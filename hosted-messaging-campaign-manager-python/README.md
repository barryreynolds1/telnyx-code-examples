# Hosted Messaging Campaign Manager

## What Does This Example Do?

Manage hosted messaging campaigns with subscriber opt-in/out tracking, delivery analytics, and TCPA-compliant STOP/START handling.

## Prerequisites

- Python 3.8+
- Telnyx account with API key from [portal.telnyx.com](https://portal.telnyx.com)

## Quick Start

```bash
git clone https://github.com/team-telnyx/telnyx-code-examples.git
cd telnyx-code-examples/hosted-messaging-campaign-manager-python
cp .env.example .env
# Edit .env with your credentials
make setup && make run
```

## Products Used

| Product | Role |
|---------|------|
| Hosted Messaging | Campaign management |
| SMS | Subscriber messaging |
| Webhooks | Delivery tracking |
