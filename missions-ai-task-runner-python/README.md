# Missions AI Task Runner

## What Does This Example Do?

AI-driven task execution within the Missions framework. Give an objective, AI plans steps and executes Telnyx API actions autonomously.

## Prerequisites

- Python 3.8+
- Telnyx account with API key from [portal.telnyx.com](https://portal.telnyx.com)

## Quick Start

```bash
git clone https://github.com/team-telnyx/telnyx-code-examples.git
cd telnyx-code-examples/missions-ai-task-runner-python
cp .env.example .env
# Edit .env with your credentials
make setup && make run
```

## Products Used

| Product | Role |
|---------|------|
| Missions API | Task framework |
| Inference | AI planning + execution |
